Work Experience

HTML Developer (2018-2019)
An HTML developer is a specialist who designs and builds web pages using HTML. They create the structure and layout of websites, ensuring they are user-friendly and visually appealing. HTML developers often work alongside designers and use additional technologies like CSS and JavaScript to enhance website functionality and interactivity.
CSS Developer (2019-2020)
A CSS developer specializes in creating stylesheets to control the visual appearance and layout of web pages. They work with HTML and JavaScript to ensure consistent design across a website, focusing on aspects like typography, colors, spacing, and responsive design for various devices. CSS developers play a crucial role in crafting user-friendly and visually appealing web experiences.
JavaScript Developer (2020-2021)
A JavaScript developer is a professional who specializes in coding and implementing JavaScript, a programming language used for web development. They create interactive and dynamic elements on websites, such as animations, forms, and user interfaces. JavaScript developers work closely with HTML and CSS to enhance the functionality and user experience of web applications.
Web Designer (2021-2022)
A web designer developer is a versatile professional who combines skills in web design and development. They create visually appealing layouts and user interfaces using tools like Adobe Photoshop and Illustrator. Additionally, they implement these designs into functional websites using HTML, CSS, and JavaScript. Web designer developers ensure that websites are both aesthetically pleasing and technically robust, delivering engaging user experiences.
Graphic Designer (2022-2023)
graphic designer is a creative professional who specializes in visual communication. They use various tools and software like Adobe Photoshop, Illustrator, and InDesign to create designs such as logos, brochures, advertisements, and website graphics. Graphic designers work closely with clients to understand their needs and create visually compelling solutions that effectively convey messages and concepts.
Frontend Developer (2023-2024)
A frontend developer is a professional who specializes in creating the user-facing side of websites and web applications. They work with technologies like HTML, CSS, and JavaScript to build interactive and visually appealing interfaces that users can interact with directly. Frontend developers collaborate closely with designers and backend developers to ensure a seamless and functional user experience across all devices.
